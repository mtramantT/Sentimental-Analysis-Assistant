Directives!

A directive is a way that we can invoke angular functionality from within an html file. They act as syntax that can be used in the html fromat without breaking the front end.

Structural Directives

*ngIf

What does the * mean in front of ngIf???

Any directive that has an asterisk(*) menas that it is a "structural directive." This means that the directive in question will actually manipulate the DOM

Attribute Directives

Attribute directives are directives tat are applied as attributes in the elements themselves, like structural directives. However, these directives do NOT aim to change the actual structure of the html, rather just the style
